## Edge Cases
- `.env` fallback for Steam shared/identity secrets, Steam cookies, and UUYP token.
- Missing UUYP token, missing Steam cookies, missing identity secret, network timeout, non-JSON response, non-zero UUYP code.
- Duplicate `account_id + order_no + tradeofferid + direction` after process restart.
- Token, cookie, identity secret, and trade URL fixture strings must not appear in errors, event details, tick summaries, or API responses.

- Stage 2: UUYP order evidence -> UUYP-owned offer executor -> Steam accept/confirm helper.
- Stage 3: purchase queue tick -> durable UUYP action ledger -> non-blocking state transition -> deduped follow-up tick.
