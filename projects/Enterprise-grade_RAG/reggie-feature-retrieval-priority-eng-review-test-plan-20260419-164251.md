## Affected Pages/Routes
- `scripts/agent_flow.py` CLI surface, verify `list`, `describe`, `run`, `--json`, and unknown-flow failures stay stable.
- `make agent-flow-list`, verify the Makefile wrapper still exposes the same fixed flow registry the CLI reports.
- `make agent-flow FLOW=<id>`, verify valid flow passthrough and invalid flow failure shape.

## Key Interactions to Verify
- Agent asks for available flows, receives the same 4 ids every time.
- Agent requests one flow description, receives the right make target, label, and command doc.
- Agent requests an unknown flow, receives a stable user-facing error instead of a traceback.

## Edge Cases
- Unknown flow id in CLI mode.
- `--json` output shape for `list` and `describe`.
- Wrapper drift, Makefile help and wrapper still point at the same CLI contract.

## Critical Paths
- Discover fixed flows -> choose one -> run it through the shared tool surface.
- Invalid flow selection -> clear error -> no accidental fallback to another target.
