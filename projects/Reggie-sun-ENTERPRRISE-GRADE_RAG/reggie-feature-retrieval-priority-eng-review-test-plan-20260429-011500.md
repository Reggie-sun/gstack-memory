- A wrapper should not silently drift and keep referencing an old section name or stale control-plane ownership wording.
- If `AGENTS.md` is shortened aggressively, the final report contract and source-priority rules must still remain explicit.

## Critical Paths
- Rewrite `AGENTS.md` top-level structure, then verify source ownership still resolves cleanly across `AGENTS.md`, `coding-rules.md`, and repo-local wrappers.
