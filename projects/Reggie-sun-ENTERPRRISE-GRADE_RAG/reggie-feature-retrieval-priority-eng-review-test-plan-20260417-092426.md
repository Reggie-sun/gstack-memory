## Affected Pages/Routes
- `/portal` — verify chat layout, transcript, composer, source drawer, and external source rendering
- `/portal/library` — verify document list, filter, preview, and empty state behavior
- `/portal/sop` — verify generation, preview, final rendering, and download states
- `/admin-console` — verify admin shell, nav, summary cards, and page transitions

## Key Interactions to Verify
- Submit a chat question and inspect source reveal behavior on `/portal`
- Open a history item and verify answer/source state replay on `/portal`
- Open a document preview from list state on `/portal/library`
- Generate or open an SOP and verify preview/final state transitions on `/portal/sop`
- Switch admin navigation sections on `/admin-console`

## Edge Cases
- External sources present but citations absent on `/portal`
- Empty results in document center on `/portal/library`
- No SOP selected or preview load failure on `/portal/sop`
- Narrow viewport layout on `/portal` and `/admin-console`

## Critical Paths
- Chat ask -> answer render -> source inspection
- Document selection -> preview modal open
- SOP selection/generation -> preview/final output
- Admin shell load -> section navigation
